dbmovie记录

发表于 2019-10-02 分类于 node.js

流程

  1. 抓取数据
  2. 数据入库
  3. 启动服务
  4. 渲染数据

依赖

爬取导入库

read.js进行读取操作

  • 通过request抓取了html代码
  • cheerio将html转成了dom
  • 将需要的内容存在数组(名称|评分|地址|图片|id)
  • 返回结果数组并导出read方法

db.js 连接数据库，导出promisify query 方法

write.js 将数据写入数据库

index.js为入口函数

前端展示

使用 ejs 模板，查询到数据后渲染到 index.html 页面。
