进阶命令

发表于 2019-08-21 分类于 linux

df

查看磁盘空间，-h表示以可读性较高的形式展示大小

free

查看内存的使用情况，-m 表示以 MB 为单位查看

date

  1. %F：表示完整的年月日

  2. %T：表示完整的时分秒

  3. %Y：表示四位年份

  4. %m：表示两位月份（带前导0）

  5. %d：表示日期（带前导0）

  6. %H：表示小时（带前导0）

  7. %M：表示分钟（带前导0）

  8. %S：表示秒数（带前导0）
