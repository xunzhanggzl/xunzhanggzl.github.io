第七章 面向对象与原型

理解原型

在 JavaScript 中，对象的原型属性是内置属性（使用标记[[prototype]]），无法直接访问。内置方法 Object.setPrototypeOf 需要传入两个对象作为参数，并将第二个对象设置为第一个对象的原型。

每个对象都可以有一个原型，每个对象的原型也可以拥有一个原型，以此类推，形成一个原型链，查找特定属性将会被委托在整个原型链上，只有当没有更多的原型可以进行查找时，才会停止查找。
